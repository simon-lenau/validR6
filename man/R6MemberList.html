R6MemberListR Documentation

R6MemberList: Wrapping toolbox for using R6Members in R6Class definitions.

Description

A R6MemberList provides utility methods for declaring R6Classes with members wrapped in R6Member objects. As such, the R6MemberList encloses zero or more R6Member objects that define the members in a R6Class.
The core method for this purpose is $R6Class().

Details

Once a R6MemberList is instanciated, the $R6Class() method can be used to define a R6Class and delegate the enclosed R6Member objects as members to this new R6Class.

The core method for this purpose is $R6Class(), which calls the worker methods $public_list(), $private_list() and $active_bindings(). These set up the lists defining the arguments public, private and active for the call to R6::R6Class().

Retrieving private and public names

Use $names() to retrieve public or private member names of the R6Member objects in the R6Class created by $R6Class().

Subsetting

The $select() method allows access to the enclosed R6Member objects by name. Alternatively, the S3-operators [ and [[ allow subsetting for R6MemberList as well.

See the methods and examples section for more details and examples.

Examples

# ===================== > Create R6MemberRestrictions < ====================== #

# Restriction that checks if length of value equals 2
length_restriction <-
    R6MemberRestriction$new(
        id = "Length check",
        restriction = function(x) {
            if (length(x) != 2) {
                stop("Length should be 2!")
            }
            return(TRUE)
        }
    )

# Restriction that checks if type of value is "character"
type_restriction <-
    R6MemberRestriction$new(
        id = "Type check",
        restriction = function(x) {
            if (typeof(x) != "character") {
                stop("type should be character!")
            }
            return(TRUE)
        }
    )

# =========================== > Create R6Members < =========================== #

# R6Member object without restriction(s):
#   may take any value
R6Member_any <-
   R6Member$new(
       name = "example_member_any",
       value = 1:5
   )

# R6Member object with restriction(s):
#   all restriction(s) check(s) must return TRUE
R6Member_two_string <-
   R6Member$new(
       name = "example_member_char_len2",
       value = c("foo", "bar"),
       restrictions = list(
               "is length 2" = length_restriction,
               "is character" = type_restriction
       )
   )

# ========================= > Create R6MemberList < ========================== #

R6MemberList_example <-
   R6MemberList$new(
       # R6Member object without restrictions
       R6Member_any,
       # R6Member object with restriction(s)
       R6Member_two_string
   )

Method R6Class()

Declare R6Class from R6MemberList

Usage
R6MemberList$R6Class(classname, print_method = "default")
Arguments
classname

Name of the R6Class to be defined. See also the documentation of argument 'classname' in ?R6Class.

Type:   character         

print_method

The print method to use for the R6Classes to be defined. May be one of the following:

"none" or NULL No own print method is specified. The parent class' print method is inherited.
"default"              A default print method is used. 
function The provided function will be used as print method.
Type:   character, function or NULL (See above.)
Default:  "default"       

The "default" print method
takes the same arguments asR6Member$print(), namely value_level, restriction_level and print_class .
It generates output like in the following example: 


Object of class(es)
   `"ExampleR6Class"`
Members:
R6Member example_member_any
   Value:
      `1:5` 
R6Member example_member_char_len2
   Value:
      `c("foo", "bar")`
Returns

A R6Class with members defined by the R6MemberList

Method select()

Subsetting method for R6MemberList

Usage
R6MemberList$select(member, new_value, inner = TRUE)
Arguments
member

Names of the R6Member objects in the R6MemberList to return.

Type:   character         

new_value

Either missing (for the getter), or the new value(s) to be assigned to the R6Member objects in the R6MemberList (setter case).

Type:   any          

inner

If TRUE, the subset is a single R6Member object from the R6MemberList, in analogy to list[[elem]].
If FALSE, the subset itself is a R6MemberList containing R6Member objects, in analogy to list[elems].

Type:   logical         
Default:  TRUE        

Returns

select
